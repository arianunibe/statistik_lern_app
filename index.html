Statistik Lern-App

Interaktive Prüfungsvorbereitung — Theorie, KI-Aufgaben, Methoden-Finder und Formelsammlung in einer App.

Was steckt drin?
📖
Theorie
8 Kapitel mit 34 Themen — von deskriptiver Statistik bis Regression. Jedes Thema enthält strukturierte Erklärungen mit LaTeX-Formeln. Wähle links in der Sidebar ein Thema aus.
✏
Übungsaufgaben
KI-generierte Aufgaben für jedes Thema — Berechnung, Multiple Choice, Interpretation oder gemischter Mix. Nach jeder Aufgabe kannst du angeben ob du sie verstanden hast; schwache Themen erscheinen in der Sidebar unter "Priorität: Nochmal üben".
🧭
Methoden-Finder
Entscheidungsbaum: Beantworte Schritt für Schritt Fragen zu Messniveau, Anzahl Variablen und Verteilung — die App führt dich zur passenden Methode. Dazu: KI-generierte Forschungsszenarien zum Trainieren der Methodenwahl.
📋
Formelsammlung
Alle Kernformeln auf einen Blick: Notation-Tabelle (Stichprobe / Population / Schätzer), Lagemasse, Streuungsmasse, Normalverteilung, Konfidenzintervalle, Signifikanztests, Zusammenhangsmasse und Regression. Mit Suchfunktion und Legenden.

API-Key einrichten — Die KI-Funktionen (Übungsaufgaben & Szenario-Training) benötigen einen persönlichen Anthropic API-Key. Zur Anleitung →

🔑 API-Key einrichten

Die Übungsaufgaben und das Szenario-Training werden von der KI Claude (Anthropic) generiert. Dafür brauchst du einen persönlichen API-Key – kein Abo, nur pay-as-you-go.

1
Konto erstellen

Geh auf console.anthropic.com und registriere dich mit deiner E-Mail-Adresse.

2
API-Key erstellen

Nach dem Login: linkes Menü → API Keys → Create Key. Gib dem Key einen Namen (z.B. "Statistik App").

3
Key kopieren

Der Key wird nur einmal angezeigt und beginnt mit sk-ant-api03-…. Kopiere ihn sofort.

4
Key in der App eintragen

Füge den Key in das Feld 🔑 API-Key unten links in der Sidebar ein und klicke OK. Der Key wird lokal in deinem Browser gespeichert und nie übertragen.

5
Ausgabelimit setzen (empfohlen)

Auf console.anthropic.com unter Billing → Usage limits kannst du ein monatliches Maximum setzen (z.B. $10), damit keine unerwarteten Kosten entstehen.

💰 Was kostet es?

Die App verwendet Claude Haiku, das günstigste Modell. Bei täglichem Lernen von 1–2h (ca. 20–40 Aufgaben/Tag) entstehen Kosten von etwa $0.15–0.25 pro Tag bzw. $4–8 pro Monat.

Der Key bleibt in deinem Browser gespeichert (localStorage) – auf einem neuen Gerät musst du ihn einmal neu eingeben. Er wird nie an Dritte weitergegeben.
